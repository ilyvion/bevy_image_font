Expand description

System that renders each ImageFontText as child [Sprite] entities, where each sprite represents a character in the text. Each sprite is positioned based on its order in the text, accounting for letter spacing, scaling mode, and anchor alignment. This system only runs when the ImageFontText or ImageFontSpriteText changes.
